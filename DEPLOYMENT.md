## ✨ Features

- **Dark-to-Light Storytelling Journey**: From pain to clarity to strength
- **Scroll Progress Indicator**: Track your position in the story
- **Section Navigator**: Jump to different emotional chapters
- **Smooth Animations**: Framer Motion-powered transitions
- **Interactive Elements**: Tap-to-reveal truth sections
- **Responsive Design**: Works beautifully on desktop and mobile
- **Symbolic Visuals**: Non-graphic, emotionally immersive storytelling

## 🚀 Local Development

```bash
# Clone the repository
git clone https://github.com/Deep23Lab/TestDN.git
cd TestDN

# Install dependencies
npm install

# Run development server
npm run dev

# Open http://localhost:3000
```

---

## 📦 Build & Deploy

```bash
# Build for production
npm run build

# Test production build locally
npm start

# Deployment is automatic:
# - Push to main branch → Actions builds and deploys to gh-pages
```
